# Cocina Argentina

Una página web con recetas argentinas tradicionales, construida con React, Vite, Tailwind CSS y Framer Motion.

## Características

- Lista de recetas con búsqueda
- Detalles completos de cada receta
- Diseño responsive y moderno
- Animaciones suaves
- Iconos y navegación intuitiva

## Tecnologías

- React 19
- Vite
- Tailwind CSS 4
- React Router DOM
- Framer Motion
- React Icons

## Instalación

1. Clona el repositorio
2. Instala dependencias: `npm install`
3. Ejecuta el servidor de desarrollo: `npm run dev`
4. Abre http://localhost:5173

## Construcción

Para producción: `npm run build`

## Despliegue

Puedes desplegar en Vercel, Netlify o cualquier plataforma que soporte SPA.
